# Noclip AT

Noclip AT is a Geometry Dash Geode mod that turns on noclip automatically after a percent you choose.

## Features

- Choose the percent where noclip activates
- Play normally before that percent
- Ignore deaths after the chosen percent
- Open the menu with a keybind
- Percent text input
- ON/OFF toggle

## Default keybind

The default menu keybind is **F8**.

You can change it in:

Geode → Installed → Noclip AT → Settings → Open Noclip AT Menu

## Notice

This mod is intended for practice, showcases, testing, and learning level endings.

Do not use this mod to claim legitimate completions.
